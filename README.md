# clickup-cli
clickup cli for click up api


## Configuring your cli

```bash

clickup config --token pk_<your-personal-access-token> --team-id <your-team-id>

```

## Get all workspaces

```

clickup teams

```

## Get all spaces for your default work space

```

clickup spaces

```

## Dev mode 

```
ENV=DEV ./clickup
```
